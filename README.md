# SAAS-subscription-metrics
SaaS Subscription Metrics

Aim

To analyze SaaS subscription data and understand customer subscription patterns.

Objectives

- Analyze customer plans.
- Calculate average monthly subscription fees.
- Identify active and inactive customers.
- Understand subscription trends.

Tools Used

- Python
- Pandas
- GitHub

Outcome

The project provides insights into subscription plans, customer retention, and revenue metrics.
